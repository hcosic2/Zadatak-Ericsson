Description of the project task


A simple mobile website for viewing and playback of video content sholud be created. Task solution must contain:

  • Home page with a list/thumbnails, which are also links to video content playback
  • Page for video content playback with contols and basic information
  • Popup that contains additional information about video content
  • Navigation among listed pages

Description of the project solution


Project solution meets fully the requirements of the Client (Orderer of the project solution). By opening the website (project solution), user sees a list/thumbnails of offered videos. By clicking on the desired thumbnail, the user selects the video content which he/she wants to view. After selecting the desired content, the page for playback of the selected video content opens. On this website, there is a video, video controls, video description and "More" button. The user can play, pause and stop the video as well as turn the volume up and down. The user can also click on the "More" button to see the details of the video. In the navigation bar, there are two links "Home" and "Documentation". Clicking on the "Home", we return to the homepage of the website, while clicking on the "Documentation“, the page with the documentation of the project solutions opens. The Navigation bar is available at any time on every page and serves to navigate through the website.
